# 📊 Customer Churn Prediction & Business Analytics Platform

An end-to-end Data Analytics and Machine Learning project that analyzes customer behavior, identifies churn patterns, and predicts customers who are likely to leave a telecom service.

The project combines Python, PostgreSQL, Machine Learning, and Tableau to perform data preprocessing, business analysis, and churn prediction.

---

## 🚀 Project Overview

Customer churn is one of the biggest challenges for subscription-based businesses. This project analyzes historical customer data to discover the key factors influencing churn and builds machine learning models to predict customer attrition.
